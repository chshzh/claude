## Structural Fixes

### Cleanup script bug (found in session, fixed)

The original Cleanup script tried to read `activations` from `GET /releases/{VER}`.
The Memfault API does **not** return an `activations` field on that endpoint.
The correct approach is `GET /deployments`, filtered by version + `status == 'done'`.
Fixed in both the skill and the new agent.
