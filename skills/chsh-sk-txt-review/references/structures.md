# Structural Patterns to Avoid

## Binary Contrasts

These create false drama. State the point directly.

| Pattern | Problem |
|---------|---------|
| "Not because X. Because Y." | Telegraphed reversal |
| "It's not X, it's Y." | Predictable pivot |
| "The answer isn't X. It's Y." | Rhetorical misdirection |
| "Not X. But Y." / "not X, it's Y" | Mechanical contrast |
| "not just X but also Y" | Additive hedge |
| "stops being X and starts being Y" | False transformation arc |

**Fix:** State Y directly. Drop the negation entirely.

## Negative Listing

Listing what something is *not* before revealing what it *is*. A rhetorical striptease.

| Pattern | Problem |
|---------|---------|
| "Not a X... Not a Y... A Z." | Dramatic buildup through negation |
| "It wasn't X. It wasn't Y. It was Z." | Same structure, past tense |

**Fix:** State Z. The reader doesn't need the runway.

## Dramatic Fragmentation

Sentence fragments for emphasis read as manufactured profundity.

| Pattern | Problem |
|---------|---------|
| "[Noun]. That's it. That's the [thing]." | Performative simplicity |
| "X. And Y. And Z." | Staccato drama |
| "This unlocks something. [Word]." | Artificial revelation |

**Fix:** Complete sentences. Trust content over presentation.

## False Agency

Inanimate objects don't perform human actions. AI loves this because it avoids naming the actor.

| Pattern | Fix |
|---------|-----|
| "a complaint becomes a fix" | "The team fixed it" |
| "the decision emerges" | "We decided" |
| "the culture shifts" | "People changed behavior" |
| "the data tells us" | "The data shows X; we concluded Y" |
| "the fix was deployed" | "[Person/team] deployed the fix" |
| "the update will address" | "[Team] will address it in the update" |
| "the conversation moves toward" | "We steered toward" |

**Fix:** Name the human. If no specific person fits, use "you" or "we".

## Passive Voice

Every sentence needs a subject doing something. Passive voice hides the actor.

| Pattern | Fix |
|---------|-----|
| "X was created" | Name who created it |
| "It is believed that" | Name who believes it |
| "Mistakes were made" | Name who made them |
| "The decision was reached" | Name who decided |
| "This was overlooked" | "We overlooked this" / "I missed this" |

## Rhetorical Setups

These announce insight rather than deliver it.

| Pattern | Problem |
|---------|---------|
| "What if [reframe]?" | Socratic posturing |
| "Think about it:" | Condescending prompt |
| "Here's what I mean:" | Redundant preview |
| "And that's okay." | Unnecessary permission-granting |

**Fix:** Make the point. Let readers draw conclusions.

## Sentence Starters to Avoid

| Pattern | Fix |
|---------|-----|
| Sentences starting with What/When/Where/Which/Who/Why/How | Lead with subject or verb instead |
| Paragraphs starting with "So" | Start with content |
| "Look," as an opener | Remove |

Wh- openers become a crutch. "What makes this hard is..." → "The constraint is..."

## Rhythm Anti-Patterns

| Pattern | Fix |
|---------|-----|
| Three-item lists | Use two items or one |
| Every paragraph ends with a punchy one-liner | Vary endings |
| Em dashes | Use commas or periods — no em dashes at all |
| Stacked short punchy sentences | Combine into one |
| "Not always. Not perfectly." | Hedging disguised as reassurance |

## Narrator-from-a-Distance

Floating above the scene instead of putting the reader in it.

| Pattern | Problem |
|---------|---------|
| "Nobody designed this." | Disembodied observation |
| "This happens because..." | Lecturer voice |
| "People tend to..." | Armchair sociologist |

**Fix:** Use "you" to put the reader in the room. "You don't sit down one day and decide to..." beats "Nobody designed this."

## Before/After Examples

### Throat-Clearing + Binary Contrast
**Before:** "Here's the thing: this is hard. Not because the technology is complex. Because people are complex. Let that sink in."
**After:** "This is hard. Technology is manageable. People aren't."

### Business Jargon Stack
**Before:** "In today's fast-paced landscape, we need to lean into discomfort and navigate uncertainty with clarity."
**After:** "Move faster."

### Dramatic Fragmentation
**Before:** "Speed. Quality. Cost. You can only pick two. That's it. That's the tradeoff."
**After:** "Speed, quality, cost — pick two."

### False Agency (Professional Email)
**Before:** "The issue was escalated and a resolution was reached."
**After:** "Sarah escalated the issue. We resolved it by [specific action]."
