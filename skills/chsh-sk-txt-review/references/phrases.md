# Phrases to Remove

## Throat-Clearing Openers

State the content directly. Remove these announcement phrases.

- "Here's the thing:"
- "Here's what [X]" / "Here's why [X]" / "Here's this [X]"
- "The uncomfortable truth is"
- "It turns out"
- "The real [X] is"
- "Let me be clear"
- "I'll be honest" / "I'm going to be honest"
- "Can we talk about"
- "Here's what I find interesting"
- "Here's the problem though"

Any "here's what/this/that" construction is throat-clearing before the point. Cut it.

## Emphasis Crutches

These add no meaning. Delete them.

- "Full stop." / "Period."
- "Let that sink in."
- "This matters because"
- "Make no mistake"
- "Here's why that matters"

## Adverbs

Kill all adverbs. No -ly words. No softeners, intensifiers, or hedges.

- "really" / "just" / "literally"
- "genuinely" / "honestly" / "simply"
- "actually" / "deeply" / "truly"
- "fundamentally" / "inherently" / "inevitably"
- "interestingly" / "importantly" / "crucially"

Also cut these filler phrases:

- "At its core"
- "In today's [X]"
- "It's worth noting"
- "At the end of the day"
- "When it comes to"
- "In a world where"
- "The reality is"

## Meta-Commentary

Remove self-referential asides. The message should move, not announce its own structure.

- "Let me walk you through..."
- "In this section, we'll..."
- "As we'll see..."
- "I want to explain..."
- "But that's a conversation for another time"
- "I hope this helps!"

## Vague Declaratives

Sentences that announce importance without naming the specific thing.

- "The reasons are structural"
- "The implications are significant"
- "The stakes are high"
- "The consequences are real"
- "The situation is complex"
- "This is a challenging time"

**Fix:** Replace with the specific thing — what, who, by when.

## Performative Sincerity

False intimacy or manufactured earnestness:

- "I promise"
- "I genuinely believe"
- "I truly appreciate"
- "I sincerely apologize" → use "I apologize" — "sincerely" is the tell
- "I hope this email finds you well"
- "creeps in"

## Business Jargon (Extended)

| Avoid | Use instead |
|-------|-------------|
| Navigate (challenges) | Handle, address |
| Unpack (analysis) | Explain, examine |
| Lean into | Accept, embrace |
| Landscape (context) | Situation, field |
| Game-changer | Significant, important |
| Double down | Commit, increase |
| Deep dive | Analysis, examination |
| Take a step back | Reconsider |
| Moving forward | Next, from now |
| Circle back | Return to, revisit |
| On the same page | Aligned, agreed |
| Reach out | Contact, email, call |
| Touch base | Check in |
| Bandwidth | Capacity, time |
| Synergy | Cooperation (or just cut it) |
| Leverage (as verb) | Use |
| Action item | Task |
| Low-hanging fruit | Easy win, quick fix |
